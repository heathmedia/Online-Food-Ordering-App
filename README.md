## About

A front-end-only demo of a web app for ordering meals. Includes the following functionality
- Login: any properly formatted email and 5+ character password will do.
- Dashboard: view meals, filter by main ingredient, and add meals to your cart.
- Cart: view items added to the cart, increase/decrease their quantities, and check out.
